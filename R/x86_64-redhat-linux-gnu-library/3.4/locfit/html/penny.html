penny {locfit}R Documentation

Penny Thickness Dataset

Description

For each year, 1945 to 1989, the thickness of two U.S. pennies was recorded.

Usage

data(penny)

Format

A dataframe.

Source

Scott (1992).

References

Scott (1992). Multivariate Density Estimation. Wiley, New York.
